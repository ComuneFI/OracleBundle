Installazione:
-------------

- Aggiungere tramite composer:
```
composer require fi/oraclebundle
```
- Aggiungere nel file app/AppKernel.php nella funzione registerBundles;
```
    new Fi\OracleBundle\FiOracleBundle(),
```
